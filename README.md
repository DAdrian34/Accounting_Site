# Accounting Site

A Django-based business website built as a self-learning project for an accounting services company.

## Live Website

https://expertcontabilbrasov.ro

## What I Built

* Multi-page business website
* Responsive layout using HTML and CSS
* Django project structure with templates and static files
* Contact and service information pages
* Deployment configuration for production
* Environment-variable handling for sensitive settings

## Technologies Used

* Python
* Django
* HTML
* CSS
* PostgreSQL
* Gunicorn
* WhiteNoise

## Purpose

This project was built to improve my practical web development skills and to create a real online presence for a small business.
